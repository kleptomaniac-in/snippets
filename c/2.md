I've refactored the component to accept a `DataSourceService` interface. Now you can provide data from any source by implementing this interface.

**Key Changes:**

1. **DataSourceService Interface**: A simple contract with a `fetchData()` method
2. **Flexible Implementations**: You can create services for databases, APIs, files, or any other source

**Usage Examples:**

```java
// Option 1: Anonymous class
DataSourceService myService = new DataSourceService() {
    @Override
    public List<Map<String, Object>> fetchData() {
        // Your logic here
        return dataFromDatabase();
    }
};

// Option 2: Lambda (cleaner)
```
